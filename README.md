# Sales-Performance-Dashboard 
## Project Overview
This project analyzes sales performance using SQL and Power BI.
The objective is to identify sales trends , profitable products, valuable customers and regional performance to support business decision-making.
---
## Business Problem 
The company lacks a centralized reporting solution to monitor sales and profitablity across regions, products and customers.

This dashboard helps stakeholders:
1. Track sales trends
2. Analyze profitablity
3. Identify top customers
4. Monitor regional performance
---
## Tools Used
1. SQL
2. POWER BI
3. DAX
4. EXCEL
5. GitHub
